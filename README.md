# PawCheck｜宠物 AI 轻量机会验证

PawCheck 是面向新手宠主的 AI 就医前分流与症状记录助手。本项目用于宠物 AI 创业机会的轻量验证，不提供兽医诊断。

## 产品边界

PawCheck 仅提供非诊断性的就医前风险提示、观察记录和信息整理；高风险表现应直接联系宠物急诊或执业兽医。
